# Bug Tracker & Task Manager Web Application

This is a complete Bug Tracking and Task Management Web App built using JSP, Servlets, JDBC, and MySQL.

# Project Overview
This project helps software teams manage their daily tasks and report bugs easily.  
Users can register, login, add tasks, update tasks, delete tasks, report bugs, and check progress on dashboard.

# Why This Project?
- Used in real IT companies  
- Helps us learn JSP, Servlets, JDBC  
- Perfect for resume & interviews  
- Shows teamwork and real software development  
- Strong project for college exhibition  

# Functional Requirements
1. User Registration  
2. Login & Authentication  
3. Add Task  
4. View Tasks  
5. Update Task  
6. Delete Task  
7. Add Bug Report  
8. View Bug Reports  
9. Update Bug  
10. Dashboard  

# Future Scope
- Team-based task assignment  
- Email notifications  
- File upload in bug report  
- Role-based dashboard  
- Priority-based task sorting  
- Dark/Light mode UI
